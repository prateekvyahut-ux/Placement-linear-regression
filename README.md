# Placement-linear-regression
📊 Placement Package Prediction

🔍 Problem Statement

This project predicts the placement package (in LPA) based on a student's CGPA using Linear Regression.

---

📁 Dataset

- Feature: CGPA
- Target: Package (LPA)
- Source: [placement.csv]

---

⚙️ Steps Performed

1. Data Loading using Pandas
2. Data Visualization (Scatter Plot)
3. Train-Test Split
4. Model Training using Linear Regression
5. Prediction and Evaluation

---

📈 Results

- Model used: Linear Regression
- Insight: Students with higher CGPA tend to get higher packages

---

🛠 Tech Stack

- Python
- Pandas
- Matplotlib
- Scikit-learn

---

🚀 How to Run

pip install -r requirements.txt

python linear_regression_scratch.py
